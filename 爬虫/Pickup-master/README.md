# Pickup

##介绍
Pickup是一个自动下载图片的程序



##1.0功能
- 输入关键词自动自动下载百度图片图片（30多张)
- 修复了图片链接失效而导致程序卡死的问题
- 保存图片名字改为了“关键词+数字”，并且解决了中文名存为乱码的问题


##使用说明
python PicDownloader.py
然后输入需要查找的图片关键词，就能自动下载到pictue文件目录下
